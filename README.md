# scoop-relay-baton

Scoop bucket for [relay-baton](https://github.com/dgl1231/relay-baton).

```powershell
scoop bucket add relay-baton https://github.com/dgl1231/scoop-relay-baton
scoop install relay-baton
```

Installs the standalone Windows x64 binary from the relay-baton GitHub Release.
Manifest source of truth lives in the main repo at
[`scoop/relay-baton.json`](https://github.com/dgl1231/relay-baton/blob/main/scoop/relay-baton.json).
